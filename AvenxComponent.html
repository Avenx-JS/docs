Class: AvenxComponent

AvenxComponent(initialStateopt, computedopt, bridgesopt, templateopt, methodsopt)

AvenxComponent is the base class for all Avenx components. It provides core functionality for reactivity, template rendering, and event binding. It uses a Proxy-based state management system to automatically trigger re-renders on state changes.
